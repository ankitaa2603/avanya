## Problem Statement

Millions of SBI customers, especially senior citizens, first-time digital users, and customers from semi-urban and rural regions, still visit branches for routine banking tasks such as:

- Balance enquiries
- Statement downloads
- ATM PIN reset
- Fixed Deposit creation
- Password recovery
- Understanding YONO features

This increases:

- Branch workload
- Operational costs
- Customer wait times
- Digital adoption barriers

---

## Solution

**AVANYA** is an AI-powered digital banking companion that guides customers inside YONO using voice and step-by-step assistance.

Instead of replacing banking interfaces, AVANYA makes them easier to understand and use.

Customers can simply say:

> "Mujhe statement download karna hai"

And AVANYA guides them through the process with:

- Voice assistance
- Screen guidance
- Adaptive UI
- Personalized recommendations
